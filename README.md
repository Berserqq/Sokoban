## Требования

- CMake **3.10** или новее
- Компилятор C:
  - GCC
  - Clang
  - MSVC

## Сборка проекта

### Linux / macOS

Создайте каталог для сборки:

```bash
mkdir build
cd build
```

Сгенерируйте проект:

```bash
cmake ..
```

Соберите программу:

```bash
cmake --build .
```

Запустите игру:

```bash
./Sokoban
```
